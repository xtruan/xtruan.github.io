Fortune Cookie for Android

 

This application emulates the experience of cracking open a fortune cookie. A random fortune of approximately 300 options is displayed to the user

I integrated the Facebook SDK for Android to allow posting of fortunes from within the application to a user's Facebook.

I also used the AdMob SDK to display context-sensitive ads as a possible revenue source for this application.
